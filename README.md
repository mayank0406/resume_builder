# Resume Builder Platform

A full-stack Resume Builder application built using the MERN stack that allows users to create, customize, preview, and download professional resumes.

## 🚀 Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## ✨ Features

- User Authentication
- Create & Edit Resume
- Multiple Resume Templates
- Real-Time Resume Preview
- PDF Download Functionality
- Responsive Design
- Secure Resume Management
